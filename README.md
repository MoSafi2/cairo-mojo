In progress binding to libcairo. 
currently Under development and bugs can be expected. 
to see a working example check `examples/red_rectangle_png.mojo`.

Low-level binding can be found in `src/_cairo_binding.mojo` and the basic functions are mostly working.
Higher level API and Mojo wrappers are organized under `src/cairo_core.mojo`, `src/cairo_enums.mojo`, `src/cairo_types.mojo`, and `src/cairo_convenience.mojo`.
